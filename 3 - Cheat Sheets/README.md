# Cheat Sheets

Single-page references and quick lookups. Use these when you need fast answers — for full guides, see [`1 - General Overview/`](../1%20-%20General%20Overview/) or [`5 - Skills/`](../5%20-%20Skills/).

## How this folder is organized

Cheat sheets live as standalone files (image or markdown) at the root of this folder, plus eventual subfolders by topic when the count grows.

| Type | Format | Use when |
|------|--------|----------|
| Image cheat sheets (`.png`, `.jpg`) | Screenshot from social media or visual reference | You want a printable / glanceable version |
| Markdown cheat sheets (`.md`) | Curated text reference | You want to copy/paste content or search it |

## Contributing

When adding a cheat sheet:

1. Drop the file directly in this folder (or a topical subfolder).
2. For markdown sheets, include YAML frontmatter (`title`, `description`, `topics`).
3. For images, name them descriptively: `claude-code-shortcuts.png`, not `IMG_1234.png`.
4. Update this README's "Contents" section (add it once a few sheets exist).
