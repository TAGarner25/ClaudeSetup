# Meta Prompts

"Prompts about prompts." Use these to generate new prompts from a brief, refine existing prompts, evaluate prompt quality, or systematically explore variations.

> Status: empty — populate as patterns are collected.

## Categories to seed

- **Generators** — turn a one-line description into a full prompt template
- **Refiners** — take an existing prompt and tighten it (clarity, structure, constraints)
- **Evaluators** — score a prompt against criteria (specificity, missing context, ambiguity)
- **Variation generators** — produce N variants of a prompt for A/B testing
- **Reverse engineers** — given an output, infer the prompt that produced it
