## What's Inside Claude Resources

The Claude resource file is the most developed and covers:

- **Anthropic Documentation** — Official docs on skills, tool use, MCPs, context windows, and more
- **GitHub Repositories** — Anthropic's official repos plus community projects
- **Skills** — 35+ links to data analysis, visualization, finance, database, and productivity skills
- **Agents & Frameworks** — Agent toolkits, orchestration plugins, and automation frameworks
- **Plugins** — Official and community plugin collections for Claude Code
- **Claude Code Guides** — Best practices, templates, and mastery resources

## Status

> Claude Resources is actively maintained. The remaining resource files (Stitch, Gemini, Obsidian, Flow) are placeholders that will be populated over time.
