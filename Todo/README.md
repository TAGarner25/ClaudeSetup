# Todo

Tracking lists for ongoing work in this repository.

## Files

| File | Purpose |
|------|---------|
| [todos.md](todos.md) | The active checklist of repository tasks (organized by section: General, Resources, Cheat Sheets, Prompts, Skills, Plugins, Hooks) |
| [Instagram.md](Instagram.md) | Backlog of Instagram references / posts / tools to evaluate or capture |

## How to use

- Strike off items in `todos.md` as work completes — open a PR per logical group of items.
- When a section's checklist runs out, add new items based on what's still missing from the repo.
- `Instagram.md` is an inbound queue — process items into the appropriate folder (`2 - Resources/`, `3 - Cheat Sheets/`, `4 - Prompts/`, etc.) and remove them from the queue.
